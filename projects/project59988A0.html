    • Pomodoro Gundam

    This project is inspired by my love of Gunpla, building plastic Gundam models. There are projects out there where there are fully functional creations. I wanted to implement another component that will be generally useful to everyone. The idea was to implement a Pomodoro style timer but with a Gundam theme. This will provide the user passive feedback on when the timer is up by both the lights and the sound of the motors when the arm is “waving”.

    There are two LEDs, a red one and a green one. When the button is pressed, the green LED indicate that the timer has started counting. When it reaches the end of the count, the red LED, which is encased within the Gundam head, will turn on and the arm waving will be simulated through the servo.
